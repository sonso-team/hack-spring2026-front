# DDoS-Ninja — Frontend

Клиентская часть проекта «DDoS-Ninja», разработанного в рамках Хакатона ДГТУ Весна 2026.

Монорепозиторий с двумя приложениями: админ-панель для управления лобби и браузерная игра для участников мероприятий.

## Стек

- React + TypeScript
- Vite
- SCSS
- HTML5 Canvas (игра)
- Docker Compose
- Phaser.js

## Структура

```
hack-spring2026-front/
├── apps/
│   ├── admin-panel/   — админ-панель (порт 5173)
│   └── game/          — игра для участников (порт 5174)
├── docker-compose.yml
└── README.md
```

## Быстрый старт (Docker)

```bash
docker compose up --build
```

## Переменные окружения

Задаются в `docker-compose.yml` или в `.env` файлах приложений:

| Переменная | Описание | Значение по умолчанию |
|---|---|---|
| `API_TARGET` | URL бэкенда | `https://hack.kinoko.su` |
| `WS_TARGET` | WebSocket URL (игра) | `wss://hack.kinoko.su` |

## Приложения

### Админ-панель (`apps/admin-panel`)

Интерфейс для сотрудников на стенде: создание лобби, просмотр таблицы результатов, QR-код, рандомайзер, экспорт в CSV, управление админами.

**Экраны:** логин → главная (лобби или создание) → экран лобби (результаты, управление) → список админов (суперадмин).

### Игра (`apps/game`)

Браузерная игра «server-defender». Игрок заходит по QR-коду, вводит данные, играет, видит результат.

**Экраны:** ввод данных → модалка «как играть» → игра (Canvas) → Game Over.

## Связь с бэкендом

API-документация: `openapi.yaml` в репозитории бэкенда.

**Группы эндпоинтов:**
- `/api/auth/*` — авторизация
- `/api/admin/*` — админ-панель (лобби, результаты, админы)
- `/api/game/*` — игрок (регистрация, игровой процесс)
